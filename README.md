# ⚡ Smart Appliance Energy Monitor

AI-based Smart Appliance Energy Monitoring Dashboard using Python and Streamlit.

## Features
- CSV Upload
- Energy Consumption Analysis
- Electricity Bill Estimation
- Interactive Charts
- AI Recommendations
- Appliance Ranking

## Technologies Used
- Python
- Streamlit
- Pandas
- Plotly
- NumPy

## Files Included
- app.py
- data.csv
- requirements.txt
- REPORT.pdf
- Presentation PPT
